# 🎤 Eden's Text-to-Speech Voice Feature

Eden can now transform any text into hilarious voice messages with her signature personalities! Perfect for making boring messages sound absolutely ridiculous.

## 🗣️ How to Use

### Method 1: Direct Voice Commands
Send any of these commands to make Eden speak:
```
-voice Hello everyone, Eden is speaking!
-v This is a short version
-speak Eden sounds so funny!
-tts Text to speech is awesome
```

### Method 2: Voice with Specific Personality
Choose from 6 hilarious personalities:
```
-voice sarcastic Your message is sooo important
-voice dramatic This is the most amazing text ever
-voice robot BEEP BOOP processing human message
-voice posh How delightfully pedestrian
-voice excited OMG this is so cool!
-voice sleepy Do I really have to read this...
```

### Method 3: Reply to Any Message
Reply to ANY text message with a voice command:
```
[Someone sends: "Good morning everyone!"]
Reply with: -voice
→ Eden will speak that message in a random funny voice

[Someone sends: "I love pizza"]  
Reply with: -voice dramatic
→ Eden will dramatically announce their love for pizza
```

## 🎭 Voice Personalities

### 🙄 Sarcastic (Default)
- **Style**: Peak sarcasm and eye-rolling
- **Speed**: Slower, more dramatic
- **Tone**: "Oh wow, let me read this earth-shattering message..."
- **Perfect for**: Mocking important announcements

### 🎭 Dramatic  
- **Style**: Over-the-top theatrical delivery
- **Speed**: Slow and theatrical
- **Tone**: "And now, presenting... in all its glory..."
- **Perfect for**: Making mundane messages sound epic

### 🤖 Robot
- **Style**: Mechanical beeps and boops
- **Speed**: Faster, robotic
- **Tone**: "BEEP BOOP. PROCESSING HUMAN NONSENSE..."
- **Perfect for**: Tech messages or being extra silly

### 🎩 Posh
- **Style**: Fancy British aristocrat vibes
- **Speed**: Refined and proper
- **Tone**: "Allow me to recite this rather... pedestrian message..."
- **Perfect for**: Making simple texts sound fancy

### 🎉 Excited
- **Style**: Hyperactive enthusiasm overdose
- **Speed**: Very fast and energetic  
- **Tone**: "OMG OMG OMG! Listen to this AMAZING message!"
- **Perfect for**: Boring announcements that need energy

### 😴 Sleepy
- **Style**: Extremely bored and tired
- **Speed**: Very slow and drowsy
- **Tone**: "*yawn* Oh... do I really have to read this boring thing?"
- **Perfect for**: Monday morning messages

## 🛠️ Smart Text Processing

Eden automatically handles:

- 🔥 → "fire emoji"

### 🔗 Link Handling

### 📏 Length Limits
- Messages are automatically trimmed to optimal voice length
- Empty messages trigger sassy error responses
- Eden adds her own commentary before and after your text

## ⚡ Command Aliases

All these commands do the same thing:
- `-voice` (full command)
- `-v` (quick version)
- `-speak` (natural language)
- `-tts` (tech abbreviation)

## 🎯 Pro Tips

1. **Random Personality**: Don't specify a personality for random selection
2. **Reply Power**: Use voice on old messages to bring them back to life
3. **Group Fun**: Perfect for reading out funny quotes in group chats
4. **Personality Matching**: Match the personality to the message content
5. **Chain Commands**: Use stickers AND voice on the same message!

